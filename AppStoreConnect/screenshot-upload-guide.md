# Screenshot Upload Guide

The Codex in-app browser does not support file uploads, so upload these files manually in App Store Connect.

## iPhone

Location: iOS App Version 1.0 -> Previews and Screenshots -> iPhone -> 6.5" Display

Upload all PNG files from:

`AppStoreConnect/upload-ready/iphone`

## iPad

Location: iOS App Version 1.0 -> Previews and Screenshots -> iPad

Upload all PNG files from:

`AppStoreConnect/upload-ready/ipad`
